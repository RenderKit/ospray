-Compilation
For Windows, open a console window of MSVC 2017 build
For Linux, remember to set the compiler to g++ 6 first:
   scl enable devtoolset-6 bash
   
Then, just run cmake in the current directory
   cmake .
   make -j22
   make install
After this, the supporting packages and ospray will be installed inside apex221 folder
